# LinkMe CRM

Lightweight, mobile-first CRM prototype for small and medium businesses in Africa.

## Phase 1 Prototype

This version uses plain HTML, CSS, Bootstrap 5, JavaScript, and LocalStorage.

Open `index.html` in a browser to start. The prototype includes:

- Dashboard with KPIs, simple charts, activity, and follow-ups
- Customer list, search, filters, add/delete, profile view, and WhatsApp actions
- Lead pipeline with draggable stages
- Tasks with due dates and completion tracking
- Reports with sales, customer growth, and lead summaries
- WhatsApp Center
- Notes and basic settings
- Phase 2 dashboard with KPI context, notifications, chart range controls, today focus cards, activity feed, and follow-up summaries

Local demo data is saved under the `linkme-crm-state` LocalStorage key.
